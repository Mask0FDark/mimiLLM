# Данные

`qa_seed.jsonl` содержит исходные пары на русском языке. Команда
`python tools/make_dataset.py --seed 42` детерминированно создаёт `train.txt` и
`validation.txt`. Каждый пример отделён пустой строкой и уже содержит маркеры
`Вопрос:`/`Ответ:`; специальные BOS/EOS добавляются токенизатором при загрузке.

Можно дописывать JSON-строки с полями `question`, `answer` и необязательным
массивом `variants`. Не помещайте сюда секретные или персональные данные.
